# 部署教程与环境依赖说明

## 一、环境依赖

| 依赖 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | ≥ 18（推荐 20/22/24） | 后端运行 + 前端构建 |
| npm | ≥ 9 | 包管理 |
| 操作系统 | Linux / macOS / Windows | better-sqlite3 需可编译或用预编译包 |
| MySQL（可选） | ≥ 5.7 / 8.0 | 仅生产迁移时需要，默认使用内置 SQLite |

前端构建产物为纯静态文件，可用 Nginx / Apache / 任意静态服务器托管。

## 二、本地开发部署

### 后端
```bash
cd backend
npm install
npm start
# 首次启动自动执行 建表 + 种子数据，生成 backend/data/app.db
```
环境变量（可选，通过 shell 或 .env 传入）：
- `PORT` 服务端口，默认 3000
- `JWT_SECRET` 生产必改
- `DB_PATH` SQLite 文件路径
- `UPLOAD_DIR` 上传目录

### 前端
```bash
cd frontend
npm install
npm run dev      # 开发，自动代理 /api -> http://localhost:3000
```

## 三、生产部署

### 1. 后端（PM2 常驻）
```bash
cd backend
npm install --production
JWT_SECRET=your-strong-secret PORT=3000 pm2 start src/server.js --name project-admin-api
pm2 save
```

### 2. 前端构建
```bash
cd frontend
npm install
npm run build      # 产物输出到 frontend/dist
```

### 3. Nginx 配置示例
```nginx
server {
    listen 80;
    server_name your-domain.com;
    client_max_body_size 210m;   # 大于单文件 200MB 限制

    # 前端静态资源
    location / {
        root /var/www/project-admin/dist;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_read_timeout 300s;
    }
}
```

## 四、切换到 MySQL（可选）

项目默认使用 SQLite，零配置即可运行。若生产需要 MySQL：

1. 执行 `docs/mysql_schema.sql` 建库建表（已含默认管理员 admin/admin123）。
2. 后端数据访问层集中在 `backend/src/db/index.js`（导出 better-sqlite3 实例）。迁移时将各 `routes/*.js` 中的同步 `db.prepare().run()/get()/all()` 改写为 `mysql2` 的异步查询，或引入轻量 ORM。SQL 语句本身通用性高，主要差异：
   - `datetime('now','localtime')` → `NOW()`
   - `INSERT OR REPLACE` → `INSERT ... ON DUPLICATE KEY UPDATE`
   - `.lastInsertRowid` → `result.insertId`
3. 建议保留 SQLite 作为开发环境，MySQL 作为生产环境。

## 五、数据备份

- SQLite：直接备份 `backend/data/app.db`（及 `-wal`/`-shm`）。
- 上传附件：备份 `backend/uploads/` 目录。

## 六、常见问题

- **better-sqlite3 安装报错**：确保 Node ABI 匹配，执行 `npm rebuild better-sqlite3`。
- **上传 413 错误**：调大 Nginx `client_max_body_size`。
- **token 过期**：默认 12 小时，可用 `JWT_EXPIRES_IN` 调整。
